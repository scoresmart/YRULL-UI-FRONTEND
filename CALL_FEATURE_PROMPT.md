# WhatsApp Calling — Full Implementation (Send + Receive Calls)

## What to Build

Two calling features for the WhatsApp chat UI:

1. **Call Button (outbound)** — Phone icon in chat header. Clicking sends a WhatsApp interactive call button message to the user, prompting them to call you.

2. **Incoming Call UI (receive calls)** — When a user calls the business on WhatsApp, a ringing overlay appears in the dashboard. Staff can **Accept** (starts browser-based voice call via WebRTC) or **Reject**. If no one answers within ~45 seconds, an auto-text is sent.

---

## Backend API (Already Deployed)

**Base URL:** `https://scoresmart-automations-production.up.railway.app`

### 1. Send Call Button Message
```
POST /whatsapp/call-button
Content-Type: application/json

Body: {
  "to": "61426228261",
  "message": "Tap below to call us on WhatsApp!",
  "display_text": "Call Now",
  "ttl_minutes": 10080
}

Success 200: { "status": "sent", "to": "61426228261", "result": {...} }
```

### 2. Poll Pending Incoming Calls
```
GET /whatsapp/calls/pending

Returns: {
  "pending": [
    {
      "call_id": "wamid.abc123",
      "from": "61426228261",
      "to": "61485838059",
      "sdp": "v=0\r\no=- ...",       // WebRTC SDP offer from caller
      "sdp_type": "offer",
      "direction": "USER_INITIATED",
      "timestamp": "1710400000",
      "received_at": 1710400000.5
    }
  ],
  "count": 1
}
```
Pending calls auto-expire after 60 seconds.

### 3. Accept Call (with SDP answer)
```
POST /whatsapp/call/accept
Content-Type: application/json

Body: {
  "call_id": "wamid.abc123",
  "sdp": "v=0\r\no=- ...",           // SDP answer generated by browser WebRTC
  "sdp_type": "answer"
}

Success 200: { "status": "accepted", "call_id": "wamid.abc123", "result": {...} }
```

### 4. Reject Call
```
POST /whatsapp/call/reject
Content-Type: application/json

Body: { "call_id": "wamid.abc123" }

Success 200: { "status": "rejected", "call_id": "wamid.abc123" }
```

### 5. Hang Up Active Call
```
POST /whatsapp/call/hangup
Content-Type: application/json

Body: { "call_id": "wamid.abc123" }

Success 200: { "status": "terminated", "call_id": "wamid.abc123" }
```

### 6. Get Call History
```
GET /whatsapp/calls?limit=50&direction=USER_INITIATED

Returns array of call records.
```

### 7. Generic Call Action (low-level, optional)
```
POST /whatsapp/call/action
Body: { "call_id": "...", "action": "terminate|pre_accept|accept|reject|connect", "sdp": "...", "sdp_type": "..." }
```

## How Incoming Calls Work (Flow)

```
User taps call icon in WhatsApp
        │
        ▼
Meta sends webhook to backend ──► Backend auto-sends "pre_accept" (holds call ringing)
        │                          └► Stores call in _pending_wa_calls dict
        ▼
Frontend polls GET /whatsapp/calls/pending every 2 seconds
        │
        ▼
Incoming call detected → Show ringing overlay UI
        │
   ┌────┴────┐
   ▼         ▼
 Accept    Reject
   │         │
   │         └► POST /whatsapp/call/reject → call ends
   │
   ▼
Browser: Create RTCPeerConnection
  → Set remote SDP offer from pending call
  → Get microphone access (getUserMedia)
  → Create SDP answer
  → POST /whatsapp/call/accept with SDP answer
  → Call connects! Audio flows through Meta's servers
        │
        ▼
In-call UI (mute, hang up, timer)
        │
        ▼
Hang up → POST /whatsapp/call/hangup → call terminates
```

### 1. `src/lib/api.js` — Add ALL call API methods

Add these to the existing `whatsappApi` object:

```javascript
export const whatsappApi = {
  // ... existing sendMessage method stays unchanged ...

  async sendCallButton({ to, message, displayText }) {
    try {
      const body = { to };
      if (message) body.message = message;
      if (displayText) body.display_text = displayText;
      const response = await fetch(`${ENV.API_BASE_URL}/whatsapp/call-button`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(body),
      });
      const data = await response.json();
      if (!response.ok) throw new Error(data.error || 'Failed to send call button');
      return data;
    } catch (error) {
      toast.error(error.message || 'Failed to send call button');
      throw error;
    }
  },

  async getPendingCalls() {
    const response = await fetch(`${ENV.API_BASE_URL}/whatsapp/calls/pending`);
    const data = await response.json();
    if (!response.ok) throw new Error(data.error || 'Failed to fetch pending calls');
    return data;
  },

  async acceptCall({ call_id, sdp, sdp_type = 'answer' }) {
    const response = await fetch(`${ENV.API_BASE_URL}/whatsapp/call/accept`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ call_id, sdp, sdp_type }),
    });
    const data = await response.json();
    if (!response.ok) throw new Error(data.error || 'Failed to accept call');
    return data;
  },

  async rejectCall(call_id) {
    const response = await fetch(`${ENV.API_BASE_URL}/whatsapp/call/reject`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ call_id }),
    });
    const data = await response.json();
    if (!response.ok) throw new Error(data.error || 'Failed to reject call');
    return data;
  },

  async hangupCall(call_id) {
    const response = await fetch(`${ENV.API_BASE_URL}/whatsapp/call/hangup`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ call_id }),
    });
    const data = await response.json();
    if (!response.ok) throw new Error(data.error || 'Failed to hang up');
    return data;
  },

  async getCallHistory({ limit = 50, direction } = {}) {
    const params = new URLSearchParams({ limit: String(limit) });
    if (direction) params.set('direction', direction);
    const response = await fetch(`${ENV.API_BASE_URL}/whatsapp/calls?${params}`);
    const data = await response.json();
    if (!response.ok) throw new Error(data.error || 'Failed to fetch call history');
    return data;
  },
};
```

### 2. Create `src/components/whatsapp/IncomingCallOverlay.jsx` — Ringing + In-Call UI

This is a **new component** — a full-screen overlay that appears when an incoming call is detected.

```jsx
import { useCallback, useEffect, useRef, useState } from 'react';
import { Phone, PhoneOff, Mic, MicOff, X } from 'lucide-react';
import { cn, initialsFromName, pastelClassFromString } from '../../lib/utils';
import { whatsappApi } from '../../lib/api';
import { useContacts } from '../../lib/dataHooks';
import toast from 'react-hot-toast';

// Format wa_id for display
function formatPhone(waId) {
  if (!waId) return '';
  const cleaned = waId.replace(/\D/g, '');
  if (cleaned.startsWith('61') && cleaned.length === 11) {
    return `+61 ${cleaned.slice(2, 5)} ${cleaned.slice(5, 8)} ${cleaned.slice(8)}`;
  }
  return waId;
}

export function IncomingCallOverlay() {
  const [pendingCall, setPendingCall] = useState(null);   // {call_id, from, sdp, sdp_type, ...}
  const [callState, setCallState] = useState('idle');      // idle | ringing | connecting | active | ended
  const [muted, setMuted] = useState(false);
  const [callDuration, setCallDuration] = useState(0);

  const peerConnectionRef = useRef(null);
  const localStreamRef = useRef(null);
  const remoteAudioRef = useRef(null);
  const callTimerRef = useRef(null);
  const pollIntervalRef = useRef(null);
  const ringtoneRef = useRef(null);

  const contactsQ = useContacts();

  // Find contact name from wa_id
  const callerName = pendingCall
    ? (contactsQ.data ?? []).find((c) => c.wa_id === pendingCall.from)?.name || formatPhone(pendingCall.from)
    : '';

  // ── Poll for pending calls every 2 seconds ──
  useEffect(() => {
    const poll = async () => {
      try {
        const data = await whatsappApi.getPendingCalls();
        if (data.count > 0 && callState === 'idle') {
          setPendingCall(data.pending[0]);
          setCallState('ringing');
        } else if (data.count === 0 && callState === 'ringing') {
          // Call was cancelled by user before we answered
          setPendingCall(null);
          setCallState('idle');
          stopRingtone();
        }
      } catch {
        // Silently fail polling
      }
    };

    pollIntervalRef.current = setInterval(poll, 2000);
    return () => clearInterval(pollIntervalRef.current);
  }, [callState]);

  // ── Play ringtone when ringing ──
  useEffect(() => {
    if (callState === 'ringing') {
      playRingtone();
    } else {
      stopRingtone();
    }
  }, [callState]);

  // ── Call duration timer ──
  useEffect(() => {
    if (callState === 'active') {
      setCallDuration(0);
      callTimerRef.current = setInterval(() => {
        setCallDuration((d) => d + 1);
      }, 1000);
    } else {
      clearInterval(callTimerRef.current);
    }
    return () => clearInterval(callTimerRef.current);
  }, [callState]);

  function playRingtone() {
    try {
      // Create a simple oscillator-based ringtone
      const ctx = new (window.AudioContext || window.webkitAudioContext)();
      const oscillator = ctx.createOscillator();
      const gain = ctx.createGain();
      oscillator.connect(gain);
      gain.connect(ctx.destination);
      oscillator.frequency.value = 440;
      oscillator.type = 'sine';
      gain.gain.value = 0.3;

      // Ring pattern: 1s on, 2s off
      const ring = () => {
        oscillator.start();
        setTimeout(() => { gain.gain.value = 0; }, 1000);
        setTimeout(() => { gain.gain.value = 0.3; }, 3000);
      };
      ring();
      ringtoneRef.current = { ctx, oscillator, interval: setInterval(ring, 3000) };
    } catch {
      // Audio not available
    }
  }

  function stopRingtone() {
    if (ringtoneRef.current) {
      clearInterval(ringtoneRef.current.interval);
      try {
        ringtoneRef.current.oscillator.stop();
        ringtoneRef.current.ctx.close();
      } catch { /* ignore */ }
      ringtoneRef.current = null;
    }
  }

  // ── Accept call: create WebRTC peer connection ──
  const onAccept = useCallback(async () => {
    if (!pendingCall || callState !== 'ringing') return;
    setCallState('connecting');
    stopRingtone();

    try {
      // 1. Get microphone access
      const localStream = await navigator.mediaDevices.getUserMedia({ audio: true, video: false });
      localStreamRef.current = localStream;

      // 2. Create RTCPeerConnection
      const pc = new RTCPeerConnection({
        iceServers: [{ urls: 'stun:stun.l.google.com:19302' }],
      });
      peerConnectionRef.current = pc;

      // 3. Add local audio track
      localStream.getTracks().forEach((track) => pc.addTrack(track, localStream));

      // 4. Handle remote audio
      pc.ontrack = (event) => {
        if (remoteAudioRef.current) {
          remoteAudioRef.current.srcObject = event.streams[0];
        }
      };

      pc.oniceconnectionstatechange = () => {
        if (pc.iceConnectionState === 'disconnected' || pc.iceConnectionState === 'failed') {
          onHangup();
        }
      };

      // 5. Set remote SDP offer from the caller
      await pc.setRemoteDescription(new RTCSessionDescription({
        type: pendingCall.sdp_type || 'offer',
        sdp: pendingCall.sdp,
      }));

      // 6. Create SDP answer
      const answer = await pc.createAnswer();
      await pc.setLocalDescription(answer);

      // 7. Wait for ICE gathering to complete (or timeout after 3s)
      await new Promise((resolve) => {
        if (pc.iceGatheringState === 'complete') return resolve();
        const timeout = setTimeout(resolve, 3000);
        pc.onicegatheringstatechange = () => {
          if (pc.iceGatheringState === 'complete') {
            clearTimeout(timeout);
            resolve();
          }
        };
      });

      // 8. Send SDP answer to Meta via backend
      const finalSdp = pc.localDescription.sdp;
      await whatsappApi.acceptCall({
        call_id: pendingCall.call_id,
        sdp: finalSdp,
        sdp_type: 'answer',
      });

      setCallState('active');
      toast.success('Call connected!');
    } catch (error) {
      console.error('Failed to accept call:', error);
      toast.error('Failed to connect call: ' + (error.message || 'Unknown error'));
      cleanup();
      setCallState('idle');
      setPendingCall(null);
    }
  }, [pendingCall, callState]);

  // ── Reject call ──
  const onReject = useCallback(async () => {
    if (!pendingCall) return;
    stopRingtone();
    try {
      await whatsappApi.rejectCall(pendingCall.call_id);
      toast('Call rejected');
    } catch {
      // Ignore
    }
    cleanup();
    setCallState('idle');
    setPendingCall(null);
  }, [pendingCall]);

  // ── Hang up active call ──
  const onHangup = useCallback(async () => {
    if (!pendingCall) return;
    try {
      await whatsappApi.hangupCall(pendingCall.call_id);
    } catch {
      // Ignore
    }
    cleanup();
    setCallState('idle');
    setPendingCall(null);
    toast('Call ended');
  }, [pendingCall]);

  // ── Toggle mute ──
  const toggleMute = useCallback(() => {
    if (localStreamRef.current) {
      localStreamRef.current.getAudioTracks().forEach((track) => {
        track.enabled = muted; // Toggle: if muted, enable; if unmuted, disable
      });
      setMuted(!muted);
    }
  }, [muted]);

  // ── Cleanup WebRTC resources ──
  function cleanup() {
    stopRingtone();
    if (peerConnectionRef.current) {
      peerConnectionRef.current.close();
      peerConnectionRef.current = null;
    }
    if (localStreamRef.current) {
      localStreamRef.current.getTracks().forEach((t) => t.stop());
      localStreamRef.current = null;
    }
    clearInterval(callTimerRef.current);
  }

  // Cleanup on unmount
  useEffect(() => cleanup, []);

  // Format duration as mm:ss
  const formatDuration = (seconds) => {
    const m = Math.floor(seconds / 60);
    const s = seconds % 60;
    return `${m}:${String(s).padStart(2, '0')}`;
  };

  // Don't render if idle
  if (callState === 'idle') return null;

  const avatarCls = pastelClassFromString(pendingCall?.from || '');

  return (
    <>
      {/* Hidden audio element for remote audio playback */}
      <audio ref={remoteAudioRef} autoPlay playsInline />

      {/* Full-screen overlay */}
      <div className="fixed inset-0 z-50 flex items-center justify-center bg-black/60 backdrop-blur-sm">
        <div className="w-80 rounded-2xl bg-white p-8 text-center shadow-2xl">

          {/* Caller avatar */}
          <div className={cn(
            'mx-auto mb-4 flex h-20 w-20 items-center justify-center rounded-full text-2xl font-bold',
            avatarCls,
            callState === 'ringing' && 'animate-pulse',
          )}>
            {initialsFromName(callerName)}
          </div>

          {/* Caller name */}
          <div className="text-lg font-semibold text-gray-900">{callerName}</div>
          <div className="text-sm text-gray-500">{formatPhone(pendingCall?.from)}</div>

          {/* Call state label */}
          <div className="mt-2 text-sm font-medium">
            {callState === 'ringing' && <span className="text-green-600 animate-pulse">Incoming call...</span>}
            {callState === 'connecting' && <span className="text-yellow-600">Connecting...</span>}
            {callState === 'active' && <span className="text-green-600">{formatDuration(callDuration)}</span>}
          </div>

          {/* Action buttons */}
          <div className="mt-8 flex items-center justify-center gap-6">
            {callState === 'ringing' && (
              <>
                {/* Reject */}
                <button
                  onClick={onReject}
                  className="flex h-14 w-14 items-center justify-center rounded-full bg-red-500 text-white shadow-lg transition hover:bg-red-600 active:scale-95"
                  aria-label="Reject call"
                >
                  <PhoneOff className="h-6 w-6" />
                </button>
                {/* Accept */}
                <button
                  onClick={onAccept}
                  className="flex h-14 w-14 items-center justify-center rounded-full bg-green-500 text-white shadow-lg transition hover:bg-green-600 active:scale-95 animate-bounce"
                  aria-label="Accept call"
                >
                  <Phone className="h-6 w-6" />
                </button>
              </>
            )}

            {callState === 'connecting' && (
              <div className="flex h-14 w-14 items-center justify-center rounded-full bg-yellow-100">
                <Phone className="h-6 w-6 text-yellow-600 animate-pulse" />
              </div>
            )}

            {callState === 'active' && (
              <>
                {/* Mute toggle */}
                <button
                  onClick={toggleMute}
                  className={cn(
                    'flex h-12 w-12 items-center justify-center rounded-full shadow transition active:scale-95',
                    muted ? 'bg-red-100 text-red-600' : 'bg-gray-100 text-gray-700 hover:bg-gray-200',
                  )}
                  aria-label={muted ? 'Unmute' : 'Mute'}
                >
                  {muted ? <MicOff className="h-5 w-5" /> : <Mic className="h-5 w-5" />}
                </button>
                {/* Hang up */}
                <button
                  onClick={onHangup}
                  className="flex h-14 w-14 items-center justify-center rounded-full bg-red-500 text-white shadow-lg transition hover:bg-red-600 active:scale-95"
                  aria-label="Hang up"
                >
                  <PhoneOff className="h-6 w-6" />
                </button>
              </>
            )}
          </div>
        </div>
      </div>
    </>
  );
}
```

## Important Notes

- **Browser microphone permission required** — The browser will prompt for microphone access when staff accepts a call. HTTPS is required (localhost or deployed with SSL).
- **No new Supabase tables needed.** Pending calls are stored in backend memory. Call history comes from the backend API.
- **No auth headers needed.** Same as existing endpoints.
- **Phone number format:** Pass `contact.wa_id` directly (e.g., `61426228261`).
- **Do NOT modify the backend.** All endpoints are already deployed.
- **Polling frequency:** `GET /whatsapp/calls/pending` is polled every 2 seconds. This is lightweight (returns empty array most of the time).
- **Call timeout:** Backend auto-expires pending calls after 60 seconds. If not answered, user gets an auto-text: "Sorry we missed your WhatsApp call."
- **WebRTC:** Uses standard browser WebRTC APIs (`RTCPeerConnection`, `getUserMedia`). Audio goes through Meta's TURN/STUN servers — no direct peer connection to the user. The SDP exchange happens via the backend API, not via browser-to-browser signaling.
- **One call at a time:** The overlay takes the first pending call. Multiple simultaneous calls are not supported in this UI.
- **Use `react-hot-toast`** for notifications (already in the project).
- **Use `@tanstack/react-query`** for call history (already in the project).
- **Lucide icons used:** `Phone`, `PhoneOff`, `Mic`, `MicOff`, `PhoneIncoming`, `PhoneOutgoing` — all available in lucide-react.
